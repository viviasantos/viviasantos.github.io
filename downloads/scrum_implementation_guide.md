# Didactic Guide: Scrum Implementation

## Goal
Explain Scrum in a practical way, with a focus on learning, presentation, and portfolio use.

## Flow
1. Define the problem and goal.
2. Build the initial backlog.
3. Prioritize with MoSCoW.
4. Plan the sprint.
5. Execute with Kanban.
6. Run the Review.
7. Run the Retro.

## Frameworks
- Scrum
- Kanban
- SMART
- INVEST
- MoSCoW
- PDCA

## Expected result
- Clear goal
- Prioritized stories
- Visual board
- Validated increment
- Learning for the next cycle
